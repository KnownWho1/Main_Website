ADB

Introduction

So you are looking to uninstall default apps, or just sideload a specific app not present on the playstore? Look no further

Required Tools / Items

  • 1x Android Device (Tablet or Mobile)
  • 1x USB Cable
  • Android Debug Bridge Installation from here

Steps

Enable USB Debuggin

This step is mostly the same in all android distributions.

Go to Settings > Developer Options > USB Debugging and toggle the setting to on

If Developer Options is not visible, you can unlock it through Settings > Status/Details and tapping on Model Number 10 times

Connect Phone/Tablet to a Computer

Use a simple USB (type-c etc) to connect

Open Terminal

Navigate to the folder where adb.exe is located

run adb.exe shell to find attached devices (be sure to approve the probing on the android device)

It will take a while to load the daemon for the process but if it fails at first just launch it again

Using Package Manager

List all installed apps on the device

pm list packages

remove a single application

pm uninstall --user 0 "package_name"

the name is the full path of the app ex. com.android.google.gm (gmail), you could combine grep to find individual named apps such as

pm list packages | grep "search_querry"

Find uninstalled apps that can be recovered

cmd package list packages -u | grep "package_name"

Restore uninstalled packages

cmd package install-existing "package_name"

Install an apk manually to bypass Advanced Protection on android (sideloading)

adb install "path to .apk file"

Do NOT forget to disable Debugging After You are done in Developer Settings!
